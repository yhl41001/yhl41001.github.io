写一个小程序

使用文本编辑器编辑一个以.py结尾的文件，例如hello.py，在文件中写入如下内容：
print ('Hello World!')
保存并退出编辑器，然后在命令行中输入如下内容：
$python hello.py
运行hello.py后可以看到输出如下内容：
Hello World!
